Day 1: EP 1: THE PHREAKY WORLD OF PBX HACKING

VOIP and PBX Security

Default Creds & Remote Access

  • TTPs: Verify that all devices and systems have default credentials changed and remote admin ports are disabled.
  • Recommendation: Use complex, unique passwords for each device. Disable or firewall off WAN‑side management interfaces.

Voicemail Security

  • TTPs: Assess the security of voicemail systems to prevent directory‑harvesting or brute‑forcing PINs.
  • Recommendation: Disable remote voicemail access or enforce strong PIN complexity / lockout policies.

Call Forwarding Controls

  • TTPs: Test for unauthorized call‑forwarding setups that could exfiltrate calls.
  • Recommendation: Limit call‑forwarding capabilities to authorized accounts only; audit logs for changes.
